📊 Análise de Desempenho — Alura Store
📌 Objetivo

Este projeto tem como objetivo analisar o desempenho de quatro lojas da rede fictícia Alura Store para recomendar qual unidade deve ser vendida visando a abertura de um novo empreendimento.

A decisão foi baseada em análise de dados utilizando Python, Pandas e Matplotlib.

🛠️ Tecnologias Utilizadas

Python

Pandas

Matplotlib

Google Colab / Jupyter Notebook

📂 Base de Dados

Foram utilizados quatro arquivos CSV contendo:

Produto e categoria

Preço e frete

Data da compra

Avaliação do cliente

Tipo de pagamento

Parcelas

Coordenadas geográficas

📊 Análises Realizadas

✔ Faturamento total por loja
✔ Quantidade de produtos vendidos por categoria
✔ Produtos mais e menos vendidos
✔ Média de avaliação dos clientes
✔ Custo médio de frete
✔ Evolução do faturamento ao longo do tempo

📈 Principais Visualizações
Faturamento Total por Loja

Evolução Mensal do Faturamento

Faturamento x Avaliação Média

📌 Resultados

Após análise dos indicadores financeiros, operacionais e de satisfação do cliente, foi identificada que:

🔎 A Loja 4 apresenta o menor faturamento total da rede, mesmo possuindo o menor frete médio.

Embora não tenha a pior avaliação, seu desempenho financeiro é inferior às demais unidades e não apresenta vantagem competitiva suficiente para compensar essa diferença.

✅ Recomendação Final

Recomenda-se a venda da Loja 4, pois:

Possui o menor faturamento total

Não apresenta diferenciais estratégicos claros

As demais lojas demonstram maior equilíbrio entre receita, avaliação e volume de vendas
